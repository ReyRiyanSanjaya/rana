Dokumentasi Teknis Rana Market

Selamat datang di dokumentasi lengkap Rana Market, sebuah platform Point of Sales (POS) dan Marketplace hybrid untuk UMKM.

Versi Dokumen: 1.0.0
Terakhir Diupdate: 2026-01-03

Tentang Aplikasi

Rana Market dirancang untuk memudahkan UMKM dalam mengelola penjualan, inventaris, dan laporan keuangan, sekaligus terhubung dengan pembeli melalui aplikasi mobile marketplace.

Fitur Utama

  • POS Kasir: Pencatatan transaksi cepat dan mudah.
  • Manajemen Produk: Stok, varian, dan kategori.
  • Laporan Keuangan: Laporan laba rugi otomatis.
  • Marketplace: Aplikasi pembeli untuk memesan barang dari toko terdekat.
  • PPOB: Penjualan pulsa dan token listrik (Integrasi Digiflazz).

Panduan Navigasi

Gunakan menu di sebelah kiri untuk menavigasi topik:

  • Arsitektur Sistem: Memahami cara kerja komponen sistem.
  • Server Deployment: Panduan instalasi backend di VPS.
  • Mobile Deployment: Panduan rilis aplikasi Android ke Play Store.
  • Integrasi API: Konfigurasi Digiflazz, WhatsApp, dan Maps.
  • Maintenance: Backup data dan update sistem.
